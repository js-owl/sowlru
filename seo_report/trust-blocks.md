## Структура блоков (для дальнейшего наполнения)

### 1. Команда (`#team`)

**Цель:** Experience + Expertise.

| Поле | Статус | Источник |
|------|--------|----------|
| Фото педагогов | Есть (3 чел.) | `index.html#team`, `_img/staff/` |
| ФИО и должность | Есть | index.html#team |
| Краткая биография | Сокращённо | Развернуть с index.html#team при необходимости |
| Сертификаты МИМП | Нет на главной | **Собрать:** сканы/фото сертификатов |
| Видео-приветствие руководителя | Нет | Опционально для YouTube + embed |

### 2. Документы и безопасность (`#safety`)

**Цель:** Trustworthiness.

| Материал | Статус | Действие для владельца |
|----------|--------|------------------------|
| ИНН / ОГРНИП | Есть | — |
| Лицензия / разрешение на образовательную деятельность | Уточнить | Предоставить номер и скан, если есть |
| Санитарно-эпидемиологическое заключение | Уточнить | Скан или формулировка «соответствует СП» |
| Образец договора | Нет | PDF «Образец договора» для скачивания |
| Правила внутреннего распорядка | Нет | Краткая выжимка или PDF |
| Видеонаблюдение | Уточнить | Если есть — указать зоны и политику доступа |
| Страхование ответственности | Уточнить | Указать при наличии |

### 3. Питание (`#food`)

**Цель:** закрыть возражения родителей.

| Материал | Статус | Действие |
|----------|--------|----------|
| Описание 5-разового питания | Есть | — |
| Пример меню на день | Есть (сокращённо) | Полное 20-дневное меню — PDF или страница |
| Фото кухни / обеда | Частично | Фото своей кухни вместо stock на detsad.html |
| Поставщики продуктов | Нет | Названия при желании (усиливает доверие) |
| Меню для аллергиков | Текстом | Пример индивидуального меню |

### 4. Форма и юридика

| Элемент | Статус |
|---------|--------|
| Чекбокс согласия на ПДн | Внедрён |
| Политика конфиденциальности | `privacy.html` |
| Согласие в mail.php | Не логируется | При необходимости добавить поле `consent` в письмо |

## Чек-лист материалов от владельца сада

- [ ] Актуальные фото педагогов (портрет, единый стиль)
- [ ] Сканы дипломов/сертификатов Монтессори (для about или отдельной галереи)
- [ ] Подтверждение лицензии / СЭЗ / регистрации ИП (что применимо)
- [ ] PDF образца договора или ключевые пункты (взнос, отмена, больничный)
- [ ] Полное меню на 20 дней (таблица или PDF)
- [ ] 3–5 фото: кухня, обед, детская площадка, вход в сад
- [ ] Текст о видеонаблюдении и безопасности (если используется)
- [ ] Актуальный прайс или вилка цен (хотя бы «от … руб.») — сильно снижает отказы от звонка

## Приоритет доработок

1. **Высокий:** прайс/вилка на главной или в FAQ; фото своей кухни.
2. **Средний:** PDF меню; образец договора.
3. **Низкий:** видео руководителя; блок сертификатов с lightbox.
